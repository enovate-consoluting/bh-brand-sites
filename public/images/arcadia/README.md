# Arcadia Brand Images

Place your brand assets in this folder:

## Required Images

1. **logo.png** (or logo.gif) - Main brand logo for the hero section
   - Recommended size: 300-400px wide
   - Formats: PNG, SVG, or GIF

2. **product-1.png** - First product render image
   - Recommended size: Square format (e.g., 500x500px)
   - Will be displayed in the product section

3. **product-2.png** - Second product render image (optional)
   - Recommended size: Square format (e.g., 500x500px)
   - Will be displayed in the product section

## Optional Images

- **verify_success.png** - Custom success icon (85x85px recommended)
- **verify_failed.png** - Custom failed icon (80x80px recommended)

If these are not provided, the page will use default icons.

## Image Optimization Tips

- Use WebP format when possible for smaller file sizes
- Compress PNGs using tools like TinyPNG
- Keep file sizes under 200KB for fast mobile loading
- Ensure images are optimized for mobile viewing (80% of users are on mobile)
